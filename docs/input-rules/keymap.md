# Keymap Notes

## Confirmed Rules

- 병서 원리 기반 조합 지원
- `Ctrl`, `LCtrl`, `RCtrl`, `Shift` modifier 기반 특수 자모 지원
- `Shift + ㅁ` 기본 탑재
- 모바일에서는 자체 자판 우선

## Example Inputs

- `ㅂ + ㅅ + ㄱ + ㅜ + ㄹ -> ᄢᅮᆯ`
- `ㄱ + ㅠ + ㅏ + ㄴ -> ᄀᆎᆫ`
- `ㅌ + ㅣ + ㅑ + ㄴ -> ᄐᆙᆫ`
