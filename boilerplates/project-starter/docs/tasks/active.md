# Active Tasks

이 문서는 **현재 진행 중인 실제 작업 항목**만 관리한다.
완료된 task는 `docs/tasks/archive/`로 이동한다.

## Status

- 현재 active task를 아래 형식으로 관리한다.

## Entry Template

- `TASK-<version>-<id>`
  - Related Requirement:
  - Scope:
  - Done Criteria:
  - Status: `todo` | `in_progress` | `landed`

## Rules

1. task는 requirement를 구현 가능한 작업 단위로 쪼갠 것이다.
2. source of truth는 task이며, 회고나 운영 메모를 같이 적지 않는다.
3. task가 `landed`가 되면 iteration 종료 시 archive로 이동한다.
